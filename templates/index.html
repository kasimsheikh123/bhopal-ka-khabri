Know more

This is a Machine learning model implementation. We have created a UI for a machine learning model that can predict the price of a house in a specific location of bhopal. We have trained this machine learning model using the data we scraped from Magicbricks.com and 99 acres.com . The data that was collected is used to train as well as to test the model. The accuracy of this machine learning model is 97% and the mean squared error is 1.3 which makes our Machine learning model pretty accurate.
This project will help the people who are new to Bhopal city and want to buy a house. They can use this portal and find out the approx amount that they have to pay. If they don’t have any other idea about the price they would end up paying extra to the Brokers and would deal with a great financial loss. Our project aims to reduce the risk of such frauds. We can also improve this model by adding a login system so we can inform users about future house prices if they want to buy after some time. We will also add a chatbot in this so that user can find out more about bhopal.
